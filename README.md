# 🚢 Titanic Passenger Survival Analysis

## 📌 Project Overview

This project performs **Exploratory Data Analysis (EDA)** on the Titanic dataset to identify the key factors influencing passenger survival. Using Python and data visualization techniques, the project uncovers meaningful insights related to passenger demographics, travel class, ticket fare, family size, and survival patterns.

---

## 🎯 Objectives

- Perform data cleaning and preprocessing.
- Analyze passenger demographics.
- Explore survival trends using visualizations.
- Generate business insights from the dataset.
- Demonstrate practical data analysis skills using Python.

---

## 📂 Dataset

- **Dataset:** Titanic Dataset
- **Records:** 891 Passengers
- **Features:** 12 Columns
- **Source:** Kaggle

## 💡 Key Insights

- Female passengers had a significantly higher survival rate than male passengers.
- First-class passengers showed the highest survival probability.
- Higher ticket fares were associated with increased survival.
- Passenger age influenced survival patterns.
- Family size had an impact on survival outcomes.

---

## 📌 Business Insights

- Passenger class played an important role during evacuation.
- Gender significantly influenced survival probability.
- Ticket fare can be considered an indicator of passenger class and survival likelihood.
- Feature engineering improved the overall understanding of passenger behavior.
